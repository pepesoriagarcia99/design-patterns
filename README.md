# Patrones de diseño

Formacion en patrones de diseño.

## Introduccion

Son soluciones comunes a problemas probados.
Aplicar un patron de diseño siempre depende del contexto del proyecto.

Se clasifican en 3 grupos:
- Creacionales: Ayudan a crear objetos de manera eficiente, ocultando los detalles de su instancia.
- Comportamiento: Definen como los objetos interactuan y se comunican para cumplir tareas complejas.
- Estructurales: Enfocados en como componer clases u objetos para formar estructuras grandes y reutilizables. 

### Creacionales

El objetivo es la creacion de objetos de forma flexible.
Este patron viene a resolver estas preguntas:
- ¿Como crear objetos de manera eficiente?
- Como ocultamos la logica de su instanciacion?
- ¿Como reducir el acoplamiento de las clases?

Mejoramos la flexibilidad al cambiar el proceso de creacion y promueven el principio de responsabilidad unica.
